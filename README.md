# Qt Задачи №1-4 — COM-порт монитор + JSON

Программа для мониторинга COM-порта и работы с JSON, разработанная на **Qt/C++**.
Все 4 задачи объединены в один проект.

## Список задач

### Задача №1

**QComboBox + список COM-портов**

При запуске программа автоматически показывает доступные COM-порты с их названиями. Кнопка **"Обновить"** обновляет список.

### Задача №2

**Отправка сообщений в COM-порт**

Поле ввода + кнопка **"Отправить"**. Сообщение отправляется в выбранный порт и отображается в текстовом поле с префиксом `>>>`.

### Задача №3

**Чтение из COM-порта (таймер 10 мс)**

Большое текстовое поле `QTextEdit` для вывода данных.  
Таймер `QTimer` с интервалом **10 мс** читает данные из порта и автоматически добавляет их в поле.

### Задача №4

**Создание JSON**

Пять полей ввода (`QLineEdit`):

- Имя
- Возраст
- Город
- Email
- Телефон

По нажатию на кнопку **"Создать JSON"** данные собираются в JSON-объект и выводятся в отдельное поле.

## Сборка и запуск

### Через командную строку (Windows)

```bash
# Перейдите в папку проекта
cd C:\Users\Нуне\Stand-2026-Hakobyan

# Сгенерируйте Makefile
qmake task3.pro

# Соберите проект
mingw32-make

# Запустите программу
.\release\task3.exe

# Очистить
mingw32-make clean
```

### Через Qt Creator

1. Откройте файл `task3.pro`
2. Выберите набор **MinGW 64-bit**
3. Нажмите **Run (Ctrl+R)**

## Как проверить работу

1. **Подключите устройство** (STM32, Arduino или виртуальный COM-порт)
2. **Запустите программу**
3. Нажмите **"Обновить"** → появится список портов
4. Выберите порт → нажмите **"Подключиться"**
5. Данные из порта начнут выводиться в текстовое поле
6. Введите сообщение → нажмите **"Отправить"**
7. Заполните поля → нажмите **"Создать JSON"**
